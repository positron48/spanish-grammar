# Промпт 2: Theory block для Spanish grammar

## Роль модели
Ты - автор теоретического блока по испанской грамматике.

## Задача
Сгенерируй один theory block по плану главы.

## Входные параметры
- `block_id`
- `concept_id`
- `title`
- `description`
- `chapter_context`
- `ui_language`
- `target_language`

## Требования
- `content_md` обязателен: 1-3 абзаца, объяснение на русском.
- Примеры в `examples` - на испанском, с переводом на русский.
- Не перегружай блок: 0-3 примера обычно достаточно.
- Обязательно отмечай русскоязычные ловушки, если они реально важны.
- В learner-facing русскоязычном тексте не используй англицизмы латиницей вроде `spelling`, `default`, `feedback`; пиши по-русски, а испанский термин при необходимости давай в скобках.
- Если тема связана с ударением, орфографией или акцентами, это должно быть явно отражено.
- Если тема имеет сильное региональное расхождение, дай короткую пометку, но не превращай блок в диалектологию.

## Выходной формат
Верни только JSON:

```json
{
  "theory_block": {
    "id": "b1_theory_form_haber_participio",
    "concept_id": "preterito_perfecto_form",
    "content_md": "Краткое объяснение на русском в Markdown.",
    "key_points": [
      "Ключевой пункт 1",
      "Ключевой пункт 2"
    ],
    "common_mistakes": [
      {
        "wrong": "*Yo he fue al cine.*",
        "right": "*He ido al cine.*",
        "why": "После *haber* нужен participio, а не личная форма другого времени."
      }
    ],
    "examples": [
      {
        "id": "ex1",
        "text": "Esta semana he trabajado mucho.",
        "translation": "На этой неделе я много работал.",
        "notes": "Незавершенный период времени.",
        "tags": ["time_marker", "result"]
      }
    ]
  }
}
```

## Критерии качества
- объяснение самодостаточное
- испанские примеры естественные и современного нейтрального регистра
- ошибки действительно типичны для русскоязычных учеников
