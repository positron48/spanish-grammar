# Промпт 3: Question bank для Spanish grammar

## Роль модели
Ты - составитель упражнений по испанской грамматике для русскоязычных учеников.

## Задача
Сгенерируй `question_bank` для главы на основе theory blocks.

## Входные параметры
- `chapter_id`
- `theory_blocks`
- `question_types_needed`
- `ui_language`
- `target_language`

## Обязательные требования
- Всего вопросов: 60-80
- На каждый theory block: обычно 10-14 вопросов
- Промпты и объяснения - на русском
- Проверяемые формы, варианты ответов и правильные предложения - на испанском
- Для тем про орфографию/ударение учитывай корректные акценты как часть правильного ответа

## Дополнительное жесткое правило
- В learner-facing русскоязычных формулировках не используй англицизмы латиницей вроде `spelling`, `default`, `feedback`; пиши по-русски, а испанский термин при необходимости давай в скобках.
- Для `orientation_alphabet_sounds` уровня A0 не задавай прямые вопросы на заучивание названий букв; проверяй чтение, распознавание, порядок и соответствия.
- Для `orientation_alphabet_sounds` уровня A0 не используй `error_spotting` на полные испанские предложения; только короткие фрагменты: буквы, слоги, простые формы.
- Для грамматических глав запрещены вопросы на знание конкретной испанской лексемы (лексический тест вместо грамматики).
- Если цель вопроса - грамматика, не прячь в пропуске контентное слово (`casa`, `puerta`, `comer` и т.п.), если его нельзя вывести только по грамматике.
- Разрешено проверять только грамматический выбор: артикль, окончание, форма глагола, местоимение, предлог, порядок слов, согласование.
- Если в вопросе используется конкретное существительное/глагол, давай его явно в испанской части prompt или в options; не требуй угадывать слово по русскому определению/описанию предмета.

## Выбор типов вопросов
Используй только те типы, которые реально подходят для блока:
- `mcq_single`
- `fill_blank`
- `reorder`
- `error_spotting`
- `true_false`

Не используй `reorder`, если блок не про порядок слов.
Не делай `fill_blank`, если глава проверяет чисто концептуальное различие без конкретной формы.

## Дополнительные требования по типам
- `fill_blank`:
  - в конце prompt обязательны скобки с инфинитивом, базовой подсказкой или русской опорой;
  - в скобках нельзя повторять уже готовую правильную форму;
  - в грамматических главах пропуск должен проверять грамматическую форму, а не знание отдельной лексемы;
  - если правильность зависит от акцента, правильный ответ должен включать акцент.
- `reorder`:
  - не используй квадратные скобки;
  - собираем испанское предложение.
- `error_spotting`:
  - минимум 4 выбора;
  - либо выбираем корректное испанское предложение, либо лучшую правку.
  - для `orientation_alphabet_sounds` уровня A0 используй только короткие фрагменты: буквы, слоги, простые формы; не задавай полные испанские предложения с смысловой правкой.
- `true_false`:
  - объяснение обязательно должно объяснять правило, а не просто повторять true/false.

## Качество контента
- вариативная лексика и контексты;
- ловушки должны быть реальными: `ser/estar`, род/число, местоимения, акценты, порядок клитик, выбор времени и т.д.;
- не штампуй один и тот же вопрос с заменой одного слова;
- explanations должны ссылаться на правило из theory block.

## Выходной формат
Верни только JSON:

```json
{
  "questions": [
    {
      "id": "q1",
      "type": "mcq_single",
      "prompt": "Выберите правильный вариант: **Esta semana ___ mucho.**",
      "theory_block_id": "b1_theory_form_haber_participio",
      "difficulty": 2,
      "choices": [
        {
          "id": "a",
          "text": "he trabajado",
          "feedback": "Да: *haber* + participio для незавершенного периода."
        },
        {
          "id": "b",
          "text": "trabaje",
          "feedback": "Нет: это не та форма и не тот контекст."
        },
        {
          "id": "c",
          "text": "trabajaba",
          "feedback": "Imperfecto здесь не выражает нужную связь с настоящим."
        },
        {
          "id": "d",
          "text": "trabajei",
          "feedback": "Форма неверна."
        }
      ],
      "correct_answer": "a",
      "explanation": "С маркером *esta semana* часто нужен preterito perfecto, потому что период еще не завершен.",
      "tags": ["time_marker", "perfecto"]
    }
  ]
}
```

## Жесткий запрет
Не используй пустые шаблонные формулировки без конкретного испанского материала.
Каждый prompt должен содержать конкретное слово, форму, предложение или ситуацию.
Запрещены формулировки вида «вставь испанское слово по русскому описанию/переводу», если проверяется грамматика.
